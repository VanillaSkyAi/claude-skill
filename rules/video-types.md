# Video Type Presets

Starting recipes, not molds. Detect the video type from context and adapt. Always call `list_templates` first to see what's available.

## Detecting Video Type

| Cue | Likely type |
|-----|-------------|
| "ad", "advertisement", "commercial" | Ad |
| "trailer", "cinematic", "reveal" | Trailer |
| "showreel", "portfolio", "demo" | Showreel |
| "TikTok", "Reel", "Short", "social" | Social Media |
| "launch video", "product video" | Trailer or Ad (ask) |
| "our story", "who we are" | Brand Story |
| "announcement", "save the date" | Event / Launch |
| No clear type | Default to Trailer |

---

## Ad (15–30s)

**Goal:** Hook fast, show value, drive action.

| Aspect | Guideline |
|--------|-----------|
| Duration | 15–30s (5–6 scenes) |
| Pacing | Fast — each scene 1 beat |
| Fonts | Bold display: Bebas Neue, Anton, Oswald |
| Text effects | `slam`, `bounce-drop`, `cut-in` |

**Structure:** Hook → Problem → Solution → Proof → CTA

---

## Trailer (25–40s)

**Goal:** Build tension, tell a story, create desire.

| Aspect | Guideline |
|--------|-----------|
| Duration | 25–40s (6–10 scenes) |
| Pacing | 4-act arc: slow → rising → peak → resolve |
| Fonts | Cinematic: Bebas Neue, Cinzel, DM Serif Display |
| Text effects | `fade-in`, `zoom-through`, `word-stagger` |

**Structure:** Mood → Context → Statement → Features → Proof → Emotion → Action

---

## Showreel / Demo (30–60s)

**Goal:** Showcase breadth, demonstrate capability.

| Aspect | Guideline |
|--------|-----------|
| Duration | 30–60s (8–12 scenes) |
| Pacing | Steady rhythm, consistent energy |
| Fonts | Clean sans-serif: Inter, Montserrat, Work Sans |
| Text effects | `slide-up`, `fade-in` |

**Structure:** Statement → Work samples → Scale → Trust → Contact

---

## Social Media (15–20s)

**Goal:** Stop the scroll, one message, immediate action.

| Aspect | Guideline |
|--------|-----------|
| Duration | 15–20s (4–5 scenes) |
| Orientation | **Portrait** (9:16) |
| Pacing | Extremely fast — every second counts |
| Fonts | Extra bold: Bebas Neue, Anton |
| Text effects | `slam`, `cut-in`, `bounce-drop` |

**Structure:** Hook → Problem → Solution → CTA (max 4 words per scene)

---

## Brand Story (25–40s)

**Goal:** Build emotional connection, communicate values.

| Aspect | Guideline |
|--------|-----------|
| Duration | 25–40s (6–8 scenes) |
| Pacing | Slow, deliberate — let footage breathe |
| Fonts | Serif: Playfair Display, Cinzel |
| Text effects | `fade-in`, `word-stagger` |

**Structure:** Mood → People → Values → Impact → Trust → Brand

---

## Event / Launch (15–25s)

**Goal:** Create urgency, reveal something new.

| Aspect | Guideline |
|--------|-----------|
| Duration | 15–25s (5–6 scenes) |
| Pacing | Tease → reveal → details → urgency |
| Fonts | Bold display: Bebas Neue, Anton |
| Text effects | `slam`, `zoom-through`, `cut-in` |

**Structure:** Tease → Reveal → Context → Date/Number → CTA
